# Changelog

本文件记录 Tomato Insight 对外发布版本的主要变化。

## v1.0.0

- 完成首页、叶片检测、果实检测、项目介绍和关于我们五个页面。
- 完成叶片 10 类与果实 13 类双模型分场景检测。
- 使用 ONNX Runtime 加载叶片与果实 `best.onnx` 模型。
- 支持普通检测、叶片 SE 清晰度增强和果实 RGB 伪 NIR 增强。
- 支持 JPG、JPEG、PNG、WEBP 图片上传与格式校验。
- 完成检测框、置信度、辅助建议和历史记录展示。
- 完成果实低置信度跨场景误判控制。
- 移除视频检测功能及相关页面入口。
- 完成桌面端与移动端响应式适配。
- 完成阿里云部署及公网访问。
